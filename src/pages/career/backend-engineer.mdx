## Backend and DevOps Engineer

We build tech to understand and respond to misinformation. Our work aspires to scale civic efforts to misinformation response. This includes using machine learning to categorize and [classify multilingual and multimedia content](https://github.com/tattle-made/feluda), [detect hate speech](https://huggingface.co/tattle-admin/july22-xlmtwtroberta-da-multi?text=shut+up); creating [web scrapers](https://github.com/tattle-made/factcheck-scraper) for sourcing content from chat apps and social media in India; building [realtime multiplayer games](https://github.com/tattle-made/viral-spiral-backend), and increasing the efficiency of fact checking work. 

## About The Role

The role will include solving for the usual challenges of infrastructure at a tech agency - setting up infrastructure to facilitate quick short lived experiments and being able to support scale for one of the more public facing products. If you have a penchant for automating away infrastructure management chores and reducing AWS bills, this is the job for you.

Our Tech Stack
- Kubernetes and Docker
- AWS : EKS, EC2, S3, RDS (Postgres), Route53 and Cloudfront
- Github Actions
- Elixir, Javascript, Python

We embrace people who value their craft and if you fancy yourself as someone with a knack for systems programming and building web scale solutions, you would like working with us.

**Experiences:** 3-5 years
